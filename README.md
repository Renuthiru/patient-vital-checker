
# Patient Vital Status Checker

This is a beginner PHP project that validates patient vital signs.

## Features
- Validates Temperature, Pulse, and Blood Pressure
- Uses Higher-Order Function
- Uses Callback Functions
- Uses Recursion to scan project files

## Project Structure
- index.php → Main controller
- vitals.php → Data source
- validate.php → Validation engine
- rules.php → Logic rules
- scanner.php → Recursive file scanner

## How to Run
1. Place project inside WAMP/XAMPP htdocs
2. Open browser:
   http://localhost/patient_vital_checker/index.php

## Output
Displays patient status (HIGH / NORMAL) and project file list
